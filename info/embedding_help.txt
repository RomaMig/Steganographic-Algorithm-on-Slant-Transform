Порядок пользования ПО.
1. Выберите файл (.txt) с сообщением для встраивания.
При необходимости можете отредактировать сообщение.
2. Настройте параметры встраивания.
Эти параметры будут встроены в ключ стегоконтейнера.
Если у вас уже есть ключ (см. п.3.) этот шаг можно пропустить.
3. Откройте параметры ключа и сгенерируйте ключ.
Если у вас уже есть ключ, нажмите на кнопку "Считать ключ" и выберите файл с ключом.
Параметры встраивания автоматически считаются с ключа.
4. Если возникла необходимость изменить параметры встраивания, то после изменений
необходимо сгенерировать ключ заново, чтобы изменения закрепились за ключом
5. Выберите файл-контейнер (.png) для встраивания.
6. Нажмите кнопку "Встроить сообщение" для начала процесса встраивания.
7. Для сохранения стегоконтейнера нажмите кнопку "Сохранить стегоконтейнер".
Выберите папку, в которую будет произведено сохранение.
В данную папку поместиться каталог со стегоконтейнером и ключом к ней.
